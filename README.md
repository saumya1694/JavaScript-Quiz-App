An interactive Quiz Application built using HTML, CSS, and JavaScript that allows users to answer multiple-choice questions and receive instant feedback. The application highlights correct answers in green and incorrect ones in red, tracks the user’s score, and dynamically updates questions using JavaScript.

🚀 Features

Multiple choice quiz questions

Instant answer validation

Correct answer highlighted in green

Incorrect answer highlighted in red

Displays final score at the end

Dynamic question rendering using JavaScript

🛠️ Technologies Used

HTML

CSS

JavaScript
